== Description ==
Genera enlaces personalizados para añadir productos al carrito de WooCommerce y redirigir al carrito o al checkout según la configuración.

== Installation ==
1. Sube la carpeta del plugin a `wp-content/plugins/cart-link`.
2. Activa el plugin desde el menú de Plugins en WordPress.
3. Asegúrate de tener WooCommerce activo.
